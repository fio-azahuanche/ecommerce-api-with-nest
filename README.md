# Caos API

1. Clone project
2. ```yarn install````
3. Clone file ```.env.template``` and rename to ```.env````
4. Change environment variables
5. To set up the database ```docker-compose up -d```
6. Run ```yarn start:dev```

## Swagger

```http://localhost:3000/api#/```
